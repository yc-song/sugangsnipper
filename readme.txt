제작 및 문의: 송종현 (hyeongoon11@snu.ac.kr)
수강신청 여석을 파악하여 이메일로 발송하는 프로그램
서울대 수강편람에 맞춰져 있으므로 서울대 학우들은 바로 실행하셔서 사용하셔도 됩니다.

기본사양
Python 3.8 이상
필요 패키지: 우측 그림 참조
.exe, .apk등으로 배포 계획 없음- 소스 코드를 건드려서 다양한 상황에 맞추기가 어렵기 때문
이용시 실제로 여석이 존재하는 과목을 지정하여 코드가 제대로 작동하는지 체크해보기를 권장

향후 발전 방향: 대인원에 맞춰 병렬 실행이 가능한 서버 구축/ 카카오톡 api등을 이용한 접근성 향상
